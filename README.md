# Python Automation Portfolio

**Practical Python Applications** showcasing automation, APIs, GUI and real-world scripting skills.

## Projects Included

- **Personal Voice Assistant** — Speech recognition, task automation, Wikipedia search, weather updates
- **File Sharing System** — Client-Server architecture for local file transfer
- **Weather Forecast App** — Real-time weather data using APIs
- **Password Generator** — Secure random password creation
- **Translation App** — Text translation using APIs
- **BMI Calculator** — Simple health calculator with GUI

## Tech Stack
- **Python**
- Libraries: `speech_recognition`, `pyttsx3`, `requests`, `socket`, `tkinter`, `pywhatkit`

## How to Run Any Project
```bash
# 1. Clone the repo
git clone https://github.com/syedaashir18/python-automation-portfolio.git
cd python-automation-portfolio

# 2. Install dependencies
pip install -r requirements.txt   # (agar file nahi hai to banao)

# 3. Run specific file
python "Personal Voice Assistant.py"
